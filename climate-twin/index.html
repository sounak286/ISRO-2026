3-Tier Modular Architecture

System calculations scale independently of browser layout logic — decoupling data engineering, model inference, and presentation layers.

Tier 1 — Data Layer
Multi-Source Ingestion
GDAL Rasterio Xarray NetCDF4

Automated micro-services pull, parse, and validate native binary, NetCDF, and HDF5 formats from MOSDAC and IMD Pune servers into PostGIS.

Tier 2 — AI Engine
PINN Surrogate + Assimilation
PyTorch PyG FastAPI PostgreSQL

Physics-informed neural networks fuse conservation laws with deep learning. Ensemble Kalman filtering mitigates drift as daily observations update.

Hardware-accelerated web dashboard with Leaflet satellite/terrain basemaps, smooth heatmap overlays, wind vectors, and the What-If climatic sandbox.
